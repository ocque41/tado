### Uploading to GitHub

GitHub does not expose the social-preview image via its REST or GraphQL
API. To set it:

1. Open <https://github.com/ocque41/tado/settings>
2. Scroll to **Social preview**
3. Click **Edit** → **Upload an image…**
4. Select `assets/social-preview-v1.png`
5. Save

After upload, the image is served from
`https://repository-images.githubusercontent.com/...` and appears as
the OG card on every Tado link unfurled in X / Slack / Discord / etc.
