# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. All contributors, maintainers, and participants are expected to adhere to it.

## Reporting

If you experience or witness unacceptable behavior, please report it by emailing **hi@cumulush.com**. All reports will be reviewed and handled confidentially.

## Enforcement

Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior. They may take appropriate and fair corrective action in response to violations.

For the full text of the code of conduct, visit:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/
